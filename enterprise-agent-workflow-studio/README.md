# Enterprise Agent Workflow Studio

Enterprise Agent Workflow Studio is a Vercel-deployable Next.js case study for designing enterprise-safe agent workflows.

## What It Proves

- Enterprise-safe agent design
- Tool/API registry modeling
- Human approval gates
- Auditability
- Eval discipline

## Routes

- `/` product UI
- `/api/workflow` POST workflow definition to structured graph JSON
- `/api/report` POST workflow definition to markdown audit report
- `/api/eval` GET deterministic eval results

## Demo Mode

The MVP uses mock tools only. No real enterprise systems are called.

## Local Development

```powershell
npm install
npm run test
npm run build
npm run dev
```
