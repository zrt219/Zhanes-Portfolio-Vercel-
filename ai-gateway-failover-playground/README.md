# AI Gateway Failover Playground

AI Gateway Failover Playground is a Vercel-deployable Next.js case study for platform AI infrastructure concepts.

## What It Proves

- Unified `/api/chat` route design
- Provider adapter thinking
- Outage and rate-limit fallback simulation
- Latency and cost logging
- Request trace auditability

## Routes

- `/` product UI
- `/api/chat` POST deterministic provider routing simulation
- `/api/eval` GET deterministic eval results

## Demo Mode

The MVP uses provider mocks only. No real model provider keys are required.

## Local Development

```powershell
npm install
npm run test
npm run build
npm run dev
```
